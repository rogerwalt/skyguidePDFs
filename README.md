# skyguidePDFs
Script collection to generate nice PDFs out of the electronic VFR manual Switzerland published by Skyguide.

### Prerequisites
* latexmk
* make
* pdflatex

### How to use
Clone into current VFR manual folder (e.g. `151210`), run `make`.
